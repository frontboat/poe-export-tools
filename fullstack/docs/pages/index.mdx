## What it is

`export.tools` is a single-binary Bun web app. The server has one job: fetch a Poe
share page, pull the embedded `__NEXT_DATA__` JSON out of the HTML, and hand
back the list of attachment URLs. Everything after that happens in your
browser — rendering, previewing, and zipping.

## What it is not

- Not a Poe scraper — it only reads the HTML Poe already serves publicly.
- Not a proxy — attachments are fetched by your browser, directly from Poe's CDN.
- Not a SaaS — deploy the binary yourself in under a minute, or use the hosted
  instance at [export.tools](https://export.tools).
