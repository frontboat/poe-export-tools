# Changelog

This page tracks user-visible changes. The source of truth for release
versions is the `VERSION` string in
[`build.ts`](https://github.com/frontboat/poe-export-tools/blob/main/fullstack/build.ts)
— bump by hand when cutting a release.

## Unreleased

- Added this documentation site (vocs).

## Previous releases

For pre-doc history, see the
[git log](https://github.com/frontboat/poe-export-tools/commits/main/fullstack).
Highlights:

- `1.3.x` — Multi-stage Docker build, embedded static assets, `/api/embedded`
  and `/api/routes` debug endpoints, OG meta foundations.
- `1.2.x` — Chat view, dual JSON input formats, `next-data.json` zip entry.
- `1.1.x` — Client-side zip streaming via `client-zip`.
- `1.0.x` — Initial Bun web app extracting attachments from
  `poe.com/s/<id>` share links.
